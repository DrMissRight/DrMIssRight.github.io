A

B

如果总和sum恰好可以被3整除，则答案为0；如果总和sum取余3之后余数为2，则意味着进行2操作+1答案为1；如果总和sum取余3之后余数为1，则判断数组中是否被存在一个数取余3之后同样也等于1，如果存在只需要进行1操作将其进行删除即可，此时答案为

C

暴力枚举出 x 和 y 的值，然后计算 k 的个数即可

D

分析：

（1）若数组中的数严格递减，则 a1 % a2 % a3 % a4 = a1

（2）如果a3 % a1 = 0，也就是a3是a1的倍数；如果a4 % a1 = 0，也就是a4是a1的倍数；那么a4 % a3的值也一定是a1的倍数。

（3）所以以此类推，如果a1作为模数得不到更小值，那么其他也肯定不可以；找到最小值，如果最小值不唯一，考虑从后面搞一个比最小值还小的非0值出来。

E

分析：

（1）题意简化：$sum = a_1 + a_2 + a_3 + ….+ a_n$ 给你左区间和常量 u，你需要找到右区间 r，满足u加到(u - sum + 1)这个值最大，多个一样的位置，选择最左边的r

（2）sum > u 的时候加的都是负数了，所以我们sum最后一次小于u是临界值，我们就可以去用前缀和记录sum，然后二分求解
